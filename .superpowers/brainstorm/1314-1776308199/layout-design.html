Design Summary

  • Content panel: Fixed ~794px width (A4 at 96dpi)
  • TOC panel: ~200px, own glass panel, sticky scroll, README tab only
  • Right sidebar: Unchanged at 220px
  • Header: Byline, title, tabs stay inside the content panel
  • Non-README tabs: TOC collapses, content stays centered at 794px
  • README body: No longer uses 3-column grid — full width inside the panel with comfortable padding
  • Centering: All panels centered as a group via flex justify-content: center
