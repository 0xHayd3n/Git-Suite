What does the file explorer actually show?

The panel transforms into a file-explorer style window — but what's inside it?

A

Real repo files from GitHub

Fetches and displays the actual file tree of the repository (folders, files, etc.) — like browsing GitHub's file explorer.

B

The skill's files (local content)

Shows the files that belong to this installed skill — the .md, .json, component files etc. that are already on disk.

C

Just the aesthetic — existing content in file-explorer style

The current detail content (actions, description, skill info) is rearranged to look like a file explorer UI. No new data fetched.
