How does the transformation animate?

The header collapses and the body becomes a file tree — but how does it feel?

A

Smooth morph — everything moves at once

The header height shrinks, all header elements scale down and slide left simultaneously, while the body cross-fades from current content to the file tree. One fluid ~300ms motion.

B

Staged — body fades out first, then header collapses

Step 1: body content fades out (~150ms). Step 2: header shrinks to compact titlebar (~200ms). Step 3: file tree fades in (~150ms). Feels more deliberate.

C

Dramatic — panel collapses to a line, then expands as explorer

The whole panel squishes down to just the titlebar height, pauses a beat, then expands back up with the file tree already inside. Like a window minimize/restore.
